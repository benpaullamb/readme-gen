## Getting Started

### Prerequisites

- Python

### Installation

1. Create a Python virtual environment.

```
python3 -m venv .venv
```

2. Activate the virtual environment.

```
// On Windows
.venv/Scripts/Activate

// On Mac
source .venv/bin/activate
```

3. Install the CLI and dependencies.

```
python3 -m pip install -r requirements.txt
```
